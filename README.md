# ByMerit — The New Standard for Extraordinary Minds

ByMerit is a high-performance personal development platform designed to evaluate and cultivate top-tier strategic thinkers.

![ByMerit Premium Mockup](https://raw.githubusercontent.com/Javoputerpy/merit/main/logo.png)

## 🚀 Vision
ByMerit identifies the top 5% of cognitive agility through the **Merit Evaluation Test (MET)**, replacing traditional credentials with raw strategic horsepower.

## ✨ Premium Features
- **White-to-Black Transitions**: A sophisticated, high-contrast visual flow across all pages.
- **Wave Reveal Animations**: Dynamic SVG-style transitions for an immersive experience.
- **Rhombus Branding**: A bold, zoomed logo container that redefines the platform's identity.
- **Responsive Design**: Fully optimized for high-density displays and mobile performance.

## 🛠 Technology Stack
- **HTML5 / Vanilla CSS3**
- **JavaScript (IntersectionObserver API)**
- **Google Fonts (Outfit)**

## 📂 Project Structure
- `index.html`: Landing Experience
- `met.html`: Assessment Details
- `program.html`: Strategic Roadmap
- `team.html`: The Collective
- `about.html`: Mission & Why
